# Quiz App

## Setup

1. Install NodeJS
2. Install dependencies
    ```bash
    npm install
    ```
3. Create `.env` file in the root folder
4. Add `MONGO_DB_CONNECTING_STRING` env var
5. Add `SESSION_SECRET` env var
6. Run the application
   ```bash
   npm start
    ```
